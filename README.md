# mecenate-web

```
git clone https://github.com/battlehack-venice-mecenate/mecenate-web.git

// generate
yo meanjs

// run locally
foreman start web

// first time
heroku login
heroku create mecenate-web
git push heroku master
heroku open

// deploy heroku (webhook)
git push origin master
```
